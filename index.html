HeirCNC

HeirCNC is a AirBNB clone. Users are able to create, update, delete, and view spot listings, as well as create, edit, delete, and view spot's reviews. HeirCNC is created by using JavaScript, React.js and Redux for the frontend, as well as Express.js and PostgreSQL for the backend.

AudioDome

AudioDome was a group project created with JavaScript, React.js, Redux.js for the frontend, Python,Flask and PostgreSQL for the backend, as well as AWS to handle image and audio uploads. Users are able to upload, delete, edit, and listen to music, as well as create, update, delete, and view playlists. Once an account has been created, the user is able to edit the account, as well as delete it.

Keebsy

Keebsy is an online market place where a user can buy or sell custom mechancial keyboard related products. This application uses JavaScript, React.js, Redux.js for the frontend, Python, Flask and PostgreSQL for the backend, as well as AWS to handle image uploads. Signed in users are able to create, update, delete, and view product listings. They are also able to create, update, delete, and view any review of a product.
